2. App Description:
The billboard of dad jokes is just a simple 2 page website. The index page shows the Jokes that users can add to the site. If someone wants to add a joke, click on the Add joke link above the Billboard of dad jokes header. There is a joke suggestion provided by a dad jokes API to give users an idea of a joke they can add to the site. That's it.
